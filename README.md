# Stenor

სწრაფი, მინიმალისტური კარნახის (dictation) პროგრამა Windows-ისთვის — Wispr Flow-ის
ალტერნატივა. დაიჭირე ღილაკი, ილაპარაკე, აუშვი: ნათქვამს Google Gemini ტექსტად გარდაქმნის და
ჩაწერს იმ აპლიკაციაში, რომელიც ამ მომენტში ფოკუსშია (VS Code, Word, ბრაუზერი, Slack, Teams…).

C# / .NET 10 + WPF; ცხოვრობს სისტემურ ტრეიში (taskbar-ზე ფანჯარა არ აქვს). მოდელი:
`gemini-3.1-flash-lite`. ცივი სტარტი ≈ 1.2 წმ, უქმე რეჟიმში working set ≈ 6 MB.

## აწყობა და გამოქვეყნება

```powershell
# ჩვეულებრივი აწყობა
dotnet build Stenor.slnx -c Release

# საპროდაქშენო publish (self-contained, ReadyToRun)
dotnet publish src/Stenor.App/Stenor.App.csproj -c Release -r win-x64 --self-contained
# → src/Stenor.App/bin/Release/net10.0-windows10.0.19041.0/win-x64/publish/Stenor.exe
```

## პირველი გაშვება

პირველ გაშვებაზე პარამეტრების ფანჯარა თავად იხსნება — ჩასვი Gemini API გასაღები
<https://aistudio.google.com/apikey>-დან, სურვილისამებრ დააჭირე **Test key**-ს და შეინახე.
შემდეგ გადადი ნებისმიერ ტექსტურ ველში, **დაიჭირე მარჯვენა Ctrl**, ილაპარაკე და აუშვი —
ტექსტი ფოკუსირებულ ველში ჩაჯდება.

## პარამეტრები

ტრეის ხატულაზე ორმაგი კლიკი ან მენიუში *Settings…*:

| პარამეტრი | ნაგულისხმევი | შენიშვნა |
|---|---|---|
| Gemini API გასაღები | — | ინახება დაშიფრული (DPAPI) `%APPDATA%\Stenor\settings.json`-ში |
| ძირითადი ენა | ინგლისური | მხოლოდ მინიშნებაა ტრანსკრიფციისთვის — სხვა ენებზეც მუშაობს |
| ღილაკი (hotkey) | მარჯვენა Ctrl | ცალკეული ღილაკი (მარცხენა/მარჯვენა განირჩევა) ან კომბინაცია |
| აქტივაციის რეჟიმი | Hold | Hold = push-to-talk, Toggle = დაჭერა/დაჭერა |
| Windows-თან ერთად გაშვება | ჩართული | |

- ჩაწერა 5 წუთზე ავტომატურად ჩერდება; ოვერლეიზე ✕ აუქმებს ჩაწერას ან ტრანსკრიფციას.
- Stenor.exe-ის მეორედ გაშვება ახალ ინსტანციას არ ქმნის — მომუშავის პარამეტრებს ხსნის.
- ლოგები: `%APPDATA%\Stenor\logs\` (არასდროს შეიცავს აუდიოს, ტრანსკრიპტებს ან გასაღებს).

## ცნობილი შეზღუდვები

- **ადმინისტრატორის (elevated) ფანჯარაში** ტექსტი პირდაპირ ვერ ჯდება (UIPI) — ტრანსკრიპტი
  ბუფერში რჩება ხელით Ctrl+V-სთვის; თუ პირდაპირი ჩასმა გინდა, Stenor ადმინისტრატორად გაუშვი.
- **Win** ღილაკიანი კომბინაციებისას OS-ის მალსახმობები მაინც შეიძლება ჩაირთოს.
- ჩასმის შემდეგ ბუფერიდან მხოლოდ ტექსტი/სურათი/ფაილების სია აღდგება — აპლიკაციების კერძო
  ფორმატები იკარგება.
